Vertex AI 逆解析による Google SEO 評価研究 ― 「画像・図・表は別ロジックで加点される」説の出典調査

調査日: 2026-06-05 調査者: Claude Code（英語・日本語・中国語 Web 並列調査 + X/Grok 検索） 目的: 「海外のSEO研究で Vertex AI の仕組みを解析し、画像・図・表は本文テキストとは別ロジックで評価・加点されると主張した研究は誰のどの研究か」を特定する

0. 結論サマリー

  • 最有力の出典: Metehan Yeşilyurt（metehan.ai） による Google Discovery Engine（＝旧 Vertex AI Search） の約1年間の逆解析研究。BrightonSEO 2025年4月メインステージ登壇＋一連のブログ。
  • 「別ロジック」の技術的実体: Vertex AI / Document AI の Layout Parser が持つ「table annotation（表注釈）」「image annotation（画像注釈）」機能。表・画像は本文テキストとは別個の「チャンク」として抽出され、AI生成の説明文（アノテーション）付きで独立に評価・インデックスされる。
  • 注意: 「画像・図・表は別ロジックで加点される」という一字一句の表現を持つ単一記事は、英・日・中＋X検索でも特定できなかった。最有力候補（Metehan）の「別チャンクとして独立処理される」という技術的事実が、伝言の過程で「別ロジックで加点」という平易な表現に変換された可能性が高い。
  • 代替・混線候補: Shaun Anderson（Hobo Web、画像独立スコア imagerank）／Dan Petrovic（DEJAN AI）／Mike King（iPullRank）。

1. 最有力候補：Metehan Yeşilyurt（metehan.ai）

人物: トルコ系の独立 AI & SEO 研究者。BrightonSEO 2025年4月メインステージ登壇。 研究手法: Google Cloud の Discovery Engine（旧 Vertex AI Search）を約1年間実運用し、内部仕様から Google 検索／AI Mode の挙動を逆算。

特定したランキングシグナル（7つ）: ベース・ランキング / Gecko（埋め込み類似度）/ Jetstream（クロスアテンション）/ Keyword Matching（BM25）/ PCTR / Freshness / Boost-Bury。

「画像・図・表が別ロジック」の核心: Vertex AI / Document AI の Layout Parser に「table annotation」「image annotation」という専用処理があり、表・画像は本文テキストとは別の独立チャンクとして抽出され、AI生成の説明文付きで個別に評価・インデックスされる。これが「テキストとは別ロジックで加点される」と語られる実体。

4. 技術的背景（Google 公式一次資料）

Vertex AI / Document AI の Layout Parser は、文書を以下のように要素ごとに別チャンクとして抽出する:

要素 処理
本文テキスト（段落） 通常のテキストチャンクとして埋め込みベクトル化
表（table） テーブル本体 + AI生成アノテーション（説明文）をペアにした独立チャンク。アノテーションが検索結果への返却可否を決める
画像（image） 画像コンテンツ + AI生成キャプション・アノテーションを独立チャンク化
見出し 祖先見出し（heading ancestry）を各チャンクのメタデータに付与

公式ドキュメントの記述（要旨）:

"With table annotation, when a table is detected in a source document, a description (annotation) of the table and the table itself are assigned to a chunk. The annotation determines if the chunk should be returned in a search result."

また Vertex AI Search のマルチモーダル検索では、テキスト関連性スコアと画像類似スコアを別々に計算し Weighted RRF で合成する仕組みが公開されている。

5. 確度の整理と次アクション

「画像・図・表は別ロジックで加点される」= 誰の研究か への回答:

  • 本命: Metehan Yeşilyurt の Vertex AI / Discovery Engine 逆解析（表・画像は別チャンクで独立処理）。
  • 混線の可能性: Shaun Anderson の imagerank（画像独立スコア）の話が合体。

未解決点:

  • 「別ロジックで加点」という正確な文言を持つ単一ソースは未特定。ポッドキャスト／日本語解説動画／Xスレッド等の二次伝言が発生源の可能性。

推奨追加調査:

  1. ユーザーが元情報をどこで見聞きしたか（媒体）を確認 → 発信源をピンポイント特定。
  2. Metehan のブログ／BrightonSEO スライドの該当箇所を直接引用し、原文表現を確定。
  3. Hobo Web の imagerank 記述の精読で「画像独立スコア」説の確度を確定。
